# Supermarket Sales Data Analysis (Excel Dashboard)
This project analyzes supermarket sales data using Excel dashboards.

## Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers

## Dashboard Highlights
- Branch-wise and city-wise sales analysis
- Product line performance
- Payment method insights
- Customer type analysis

## Key Insights
- Certain product lines generate higher revenue
- E-wallet usage is increasing
- Sales performance varies by branch
